1.1 Basic Tools

We are going to start with the most basic langauge html. The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser or web page. HTML elements are the building blocks of HTML pages.HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items. HTML elements are delineated by tags, written using angle brackets

Headings:- They are only of 6 types. Basic syntax is:

<h1>,<h2>, <h3>, <h4>, <h5>, <h6>


HTML is not a Programming Language it is only a markup language

Always remember to close all tags i.e <h1></h1>

Paragraphs:- Basic syntax is:

<p>The text goes here </p>

Lists:- Divided into two:- Ordered and Ordered

Ordered:-

<ol>

<li>first item</li>

<li>second item</li>

</ol>

Unordered:-

<ul>

<li>first item</li>

<li>second item</li>

</ul>

Links:-

<a></a>

Links have href tags that direct to specific urls i.e <a href="target_url"></a>

Inputs:-

<input>

:- Are of several types:-

<input type="text"> Will render an input type for any text

<input type="number"> Will render an input type for only numbers

<input type="email"> Will render an input type for only emails

<input type="button"> Will render an input type for a button

<input type="password"> Will render an input type for a password field

<input type="hidden"> Will render an input type that is hidden from user

Dropdown Options:-

<select>

<option value="male">Male</option>

<option value="female">Female</option>

<option value="other">Other</option>

</select>
